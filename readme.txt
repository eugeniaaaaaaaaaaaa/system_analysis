В данном репозитории сохраняются задания по системному анализу (в том числе тестовые).

Файл "Согласование отпуска" содержит описание бизнес-процесса (согласования отпуска от шага "Сотрудник захотел в отпуск" до "Наступил первый день отпуска сотрудника") и техническое задание на разработку (карточка заявки на отпуск).
